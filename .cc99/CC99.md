## プロジェクト概要

**cc99** は、Anthropic の Claude Code CLI を Neovim に統合する軽量プラグイン。ThePrimeagen の「99」プラグインに着想を得ており、エディタ内でコードの選択範囲に対して Claude による生成・編集を非同期で実行できる。

## 使用言語・フレームワーク

- **Lua** — Neovim プラグイン本体の実装言語
- **Neovim API** — UI（フローティングウィンドウ）やバッファ操作
- **Claude Code CLI** — `claude --print` コマンドによる AI 連携（`Read`, `Grep`, `Glob`, `LS` ツールを許可）

## ディレクトリ構成

```
cc99/
├── .gitignore          # Lua/C ビルド成果物の除外設定
├── .luarc.json         # Lua Language Server 設定
├── LICENSE             # MIT ライセンス (2026)
├── README.md           # プロジェクト概要
├── cc99.md             # 設計ドキュメント（日本語）
└── lua/
    └── cc99/
        └── init.lua    # プラグイン本体
```

## 主要ファイル

- **`lua/cc99/init.lua`** — プラグインのコア実装。状態管理、フローティングウィンドウ UI、Claude CLI 呼び出し、コード置換ロジックを含む。キーバインドは `<leader>cco`（選択コードで窓を開く）、`<leader>ccx`（実行）、`<leader>ccq`（キャンセル）。
- **`cc99.md`** — 日本語の設計ドキュメント。既存ツール（99, claudecode.nvim, avante.nvim 等）の比較分析、設計方針（スコープ限定編集 + 非同期 + コードベース探索）、CLI フラグ設計、UI フローを記載。
- **`.luarc.json`** — `vim` 等のグローバル変数を宣言し、IDE のエラー抑制と補完を有効化。
